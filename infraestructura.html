¿Cómo se identifica un Servidor? Con las Direcciones IP

Cada dispositivo conectado a Internet tiene una identificación única en el mundo conocida como Dirección IP (Internet Protocol). Son cadenas de números como por ejemplo: 142.250.190.46.

El Sistema de Nombres de Dominio (DNS)

Como somos malísimos para recordarnos de numeros taaaan grandes, se inventó el DNS (Domain Name System). El DNS funciona como la "agenda de contactos" para Internet.

En lugar de memorizar la IP de Google, simplemente escribes google.com. Los servidores DNS se encargan de traducir ese nombre legible a la dirección IP real del servidor.

La Solicitud

  1. Escribes una dirección en tu navegador (ej. galileo.edu) y se presiona Enter.
  2. Tu computadora le pregunta a un Servidor DNS: ¿Cuál es la IP de galileo.edu?
  3. El DNS responde con la dirección IP exacta del servidor de la universidad.
  4. Nuestro navegador viaja directamente a esa IP a través de los cables de internet y le solicita los archivos visuales.
  5. El servidor responde enviando el HTML, CSS e imágenes, y tu pantalla dibuja la página web.
